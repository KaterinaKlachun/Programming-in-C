Программа является консольным приложением на языке C++.

Согласно лекциям, консольное приложение имеет входную точку `main()`, с которой начинается выполнение программы.

Программа построена как набор функций: одна функция отвечает за вывод цифр в обычном порядке, вторая --- за вывод цифр в обратном порядке.

В начале подключаются заголовочные файлы:

```
#include <stdio.h>#include <conio.h>
```

`stdio.h` подключается для использования функций консольного ввода и вывода: `printf()` и `scanf()`.\
`conio.h` нужен для функции `getch()`, чтобы окно консоли не закрылось сразу после завершения программы.

Функция:

```
void PrintNormal(int n)
```

является пользовательской функцией. По лекциям функция --- это самостоятельный блок программы, имеющий имя, список параметров и тело. Тип `void` означает, что функция не возвращает значение.

Параметр:

```
int n
```

передаёт в функцию текущее значение числа.

Внутри функции используется условный оператор:

```
if (n < 10)
```

Он проверяет базовый случай рекурсии. Если число меньше 10, значит оно состоит из одной цифры. Эту цифру можно сразу вывести:

```
printf("%d ", n);return;
```

Оператор `return` завершает выполнение функции.

Если число состоит из нескольких цифр, выполняется рекурсивный вызов:

```
PrintNormal(n / 10);
```

Операция деления на 10 отбрасывает последнюю цифру числа. Например:

```
179 / 10 = 17
```

После возврата из рекурсии выводится последняя цифра:

```
printf("%d ", n % 10);
```

Операция `%` вычисляет остаток от деления. При делении на 10 остаток является последней цифрой числа:

```
179 % 10 = 9
```

За счёт того, что сначала вызывается рекурсия, а потом выполняется вывод остатка, цифры выводятся в обычном порядке: сначала старшие, потом младшие.

Функция:

```
void PrintReverse(int n)
```

также является пользовательской функцией и работает рекурсивно. Она выводит цифры числа в обратном порядке.

Сначала снова проверяется условие:

```
if (n < 10)
```

Если число однозначное, оно выводится сразу.

Если число содержит несколько цифр, сначала выводится последняя цифра:

```
printf("%d ", n % 10);
```

Затем вызывается функция для оставшейся части числа:

```
PrintReverse(n / 10);
```

В этом случае порядок действий обратный: сначала выводится младшая цифра, потом программа переходит к следующей. Поэтому число `179` выводится как:

```
9 7 1
```

Основная функция программы:

```
int main()
```

является точкой входа. В ней объявляется переменная:

```
int n;
```

Переменная --- это поименованное место в памяти для хранения данных. В данном случае переменная `n` хранит натуральное число, введённое пользователем.

Ввод выполняется командой:

```
scanf("%d", &n);
```

`%d` используется для ввода целого числа. Символ `&` перед `n` означает передачу адреса переменной, чтобы введённое значение было записано именно в неё.

Далее выполняется проверка корректности данных:

```
if (n <= 0)
```

Так как по условию дано натуральное число, оно должно быть больше нуля. Если введено значение `0` или отрицательное число, программа выводит сообщение об ошибке и завершает работу.

После проверки вызывается функция:

```
PrintNormal(n);
```

Она выводит цифры числа в обычном порядке.

Затем вызывается:

```
PrintReverse(n);
```

Она выводит цифры числа в обратном порядке.

В конце программы используется:

```
getch();return 0;
```

`getch()` ожидает нажатия клавиши.\
`return 0` сообщает, что программа завершилась успешно.

Главная идея программы --- использование рекурсии вместо циклов. В лекциях функции рассматриваются как основной структурный элемент программы, а каждый вызов функции создаёт отдельный уровень выполнения. В данной задаче рекурсивный вызов заменяет цикл: программа многократно вызывает одну и ту же функцию, пока не дойдёт до базового случая.

В программе не используются строки, массивы и циклы. Все действия выполняются только с помощью арифметических операций:

```
n / 10
```

убирает последнюю цифру,

```
n % 10
```

получает последнюю цифру.

Общая логика работы программы:

1.  Пользователь вводит натуральное число.
2.  Программа проверяет, что число больше нуля.
3.  Для обычного порядка сначала рекурсивно обрабатывается число без последней цифры.
4.  Затем выводится последняя цифра.
5.  Для обратного порядка сначала выводится последняя цифра.
6.  Затем рекурсивно обрабатывается оставшаяся часть числа.
7.  Работа функции заканчивается, когда число становится однозначным.
