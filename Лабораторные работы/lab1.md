# Пояснение к лабораторной работе №1
## Решение квадратного уравнения на языке C

Данная программа предназначена для решения квадратного уравнения вида:

> ax² + bx + c = 0

Программа выполняет следующие действия:
- запрашивает у пользователя коэффициенты квадратного уравнения;
- вычисляет дискриминант;
- определяет количество корней уравнения;
- выводит результаты вычислений на экран.

---

## Подключение библиотек

```c
#include <stdio.h>   // printf, scanf
#include <math.h>    // pow, sqrt
#include <windows.h>
```
### Назначение библиотек
- stdio.h - Содержит функции стандартного ввода и вывода:
    - printf — вывод информации на экран; 
    - scanf — ввод данных с клавиатуры.

- math.h - Содержит математические функции:
  - pow(x, y) — возведение числа в степень; 
  - sqrt(x) — вычисление квадратного корня.

- windows.h - Используется для корректного отображения русского текста в консоли Windows.

---

### Главная функция программы
```c++
int main() {
```
Функция main является точкой входа в программу.  С неё начинается выполнение любой программы на языке C.

```c
SetConsoleOutputCP(CP_UTF8);

```
Данные функции устанавливают кодировку UTF-8 для ввода и вывода данных в консоли Windows. Это необходимо для 
корректного отображения русского текста.

### Объявление переменных
```c
double a, b, c;
double D;
double x1, x2;
```

#### Назначение переменных
- a, b, c — коэффициенты квадратного уравнения;
- D — дискриминант;
- x1, x2 — корни уравнения.

Тип данных double используется для хранения вещественных чисел, так как результаты вычислений могут быть дробными.

---

### Ввод коэффициентов с клавиатуры
```c
printf("Введите коэффициент a: ");
scanf("%lf", &a);

printf("Введите коэффициент b: ");
scanf("%lf", &b);

printf("Введите коэффициент c: ");
scanf("%lf", &c);
```

#### Пояснение
- функция printf выводит сообщение пользователю; 
- функция scanf считывает введённое значение с клавиатуры; 
- %lf — формат спецификатора для типа double; 
- символ & указывает адрес переменной в памяти.

---

### Вычисление дискриминанта
```c
D = pow(b, 2) - 4 * a * c;
```

Дискриминант вычисляется по формуле:

>  D = b² − 4ac

Функция pow(b, 2) используется для возведения коэффициента b в квадрат.

---

### Проверка значения дискриминанта
*Случай D > 0*
```c
if (D > 0) {
```

Если дискриминант больше нуля, квадратное уравнение имеет два действительных корня.

```c
x1 = (-b + sqrt(D)) / (2 * a);
x2 = (-b - sqrt(D)) / (2 * a);
```
```c
printf("D > 0\n");
printf("x1 = %.2lf\n", x1);
printf("x2 = %.2lf\n", x2);
```
---
*Случай D = 0*
```c
else if (D == 0) {
```

Если дискриминант равен нулю, квадратное уравнение имеет один действительный корень.

```c
x1 = -b / (2 * a);
```
```c
printf("D = 0\n");
printf("x = %.2lf\n", x1);
```
---
*Случай D < 0*
```c
else {
```
Если дискриминант меньше нуля, квадратное уравнение не имеет действительных корней.

```c
printf("D < 0\n");
printf("Корней нет\n");
```
---

### Завершение программы
```c
return 0;
```
Оператор return 0 завершает выполнение программы и сообщает операционной системе,
что программа завершилась корректно.
