## 1. Структуры в языке C

Структура - это пользовательский тип данных, который позволяет объединить несколько переменных различных типов в одну логическую единицу.

Структура описывает объект, состоящий из нескольких полей.

Пример объявления структуры:

```c\
struct humen {\
    char name[50];\
    char surname[50];\
    int year;\
};
```

В данном случае структура содержит:

-   строковое поле имени
-   строковое поле фамилии
-   целочисленное поле года рождения

Каждый экземпляр структуры представляет одного человека.

* * * * *

2\. Создание переменных и массивов структур
-------------------------------------------

После объявления структуры можно создавать переменные данного типа.

Пример:

```struct humen A[4];```

Это массив из четырёх элементов типа `struct humen`.

Каждый элемент массива содержит:

-   имя
-   фамилию
-   год рождения

Таким образом, массив представляет собой набор объектов одного типа.

* * * * *

3\. Обращение к полям структуры
-------------------------------

Для доступа к элементам структуры используется оператор точки.

Пример:

```
A[i].year\
A[i].name\
A[i].surname
```

Здесь:

-   `A[i]` --- i-й элемент массива
-   `.year` --- обращение к полю структуры

Данный механизм позволяет работать с отдельными характеристиками объекта.

* * * * *

4\. Ввод данных в структуру
---------------------------

Ввод значений в структуру осуществляется через функцию `scanf`.

Пример:

```scanf("%s %s %d", A[i].surname, A[i].name, &A[i].year);```

Особенности:

-   строковые поля (`char[]`) передаются без оператора `&`
-   числовые поля передаются по адресу (`&`)

Формат строки должен строго соответствовать структуре данных:\
фамилия → имя → год

* * * * *

5\. Копирование структур
------------------------

Структуры в языке C можно копировать напрямую.

Пример:

```B[i] = A[i];```

При этом:

-   копируются все поля структуры
-   не требуется отдельное копирование каждого поля

Это упрощает работу с данными по сравнению с массивами.

* * * * *

6\. Сортировка массива структур
-------------------------------

В задаче используется метод пузырька --- один из базовых алгоритмов сортировки.

Принцип работы:

-   сравниваются соседние элементы массива
-   если элементы находятся в неправильном порядке, они меняются местами
-   процесс повторяется несколько раз

Условие сравнения:

```if (B[j].year > B[j + 1].year)```

Это означает, что элементы упорядочиваются по возрастанию года рождения.

Обмен элементов:
```
struct humen temp = B[j];\
B[j] = B[j + 1];\
B[j + 1] = temp;
```

Используется временная переменная для сохранения значения.

7\. Вложенные циклы при сортировке
----------------------------------

Сортировка реализуется с помощью двух циклов:

```
for (i = 0; i < n - 1; i++) {\
    for (j = 0; j < n - i - 1; j++) {
```

Назначение:

-   внешний цикл задаёт количество проходов
-   внутренний цикл выполняет сравнение элементов

Такой подход обеспечивает корректную сортировку массива.

* * * * *

8\. Вывод данных структуры
--------------------------

Вывод осуществляется с помощью функции `printf`.

Пример:

```printf("%s %s %d\n", B[i].surname, B[i].name, B[i].year);```

Формат:

-   `%s` --- строка
-   `%d` --- целое число

Вывод производится построчно для каждого элемента массива.
