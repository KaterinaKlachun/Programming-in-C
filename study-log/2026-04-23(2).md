## 1. Что такое динамическая память

Динамическая память — это память, которая выделяется во время выполнения программы.

Используется, когда размер данных заранее неизвестен.

Выделение памяти: ```malloc(...)```

Освобождение: ```free(...)```


---

## 2. Что такое указатель

Указатель — это переменная, которая хранит адрес другой переменной.

Пример: ```int *p;```

Для матриц: ```double **A;```


---

## 3. Динамическая матрица

Создание матрицы n × n:
```
A = (double**)malloc(n * sizeof(double*));

for (i = 0; i < n; i++) {
A[i] = (double*)malloc(n * sizeof(double));
}
```

---

## 4. Операции над матрицами

Сложение: ```C[i][j] = A[i][j] + B[i][j]```

Вычитание: ```C[i][j] = A[i][j] - B[i][j]```

Умножение: ```C[i][j] = Σ(A[i][k] * B[k][j])```

## 7. Файл matrix.c

```
#include <stdlib.h>
#include "matrix.h"

double** calculate(double** A, double** B, int n, char op) {
int i, j, k;

double** result = (double**)malloc(n * sizeof(double*));

for (i = 0; i < n; i++) {
result[i] = (double*)malloc(n * sizeof(double));
}

if (op == '+') {
for (i = 0; i < n; i++)
for (j = 0; j < n; j++)
result[i][j] = A[i][j] + B[i][j];
}

else if (op == '-') {
for (i = 0; i < n; i++)
for (j = 0; j < n; j++)
result[i][j] = A[i][j] - B[i][j];
}

else if (op == '*') {
for (i = 0; i < n; i++) {
for (j = 0; j < n; j++) {
result[i][j] = 0;
for (k = 0; k < n; k++) {
result[i][j] += A[i][k] * B[k][j];
}
}
}
}

return result;

}
```
## Краткий разбор кода

- #ifndef MATRIX_H
Директива условной компиляции.
Проверяет, не был ли уже подключён этот файл ранее.

- #define MATRIX_H
Если файл ещё не подключался — создаётся определение MATRIX_H.
Это предотвращает повторное подключение файла.
Объявление функции calculate

- double** calculate(double** A, double** B, int n, char op);
объявление функции (прототип)
принимает две матрицы, размер n и знак операции
возвращает указатель на матрицу-результат

- Тип double**
```double**```
  - указатель на указатель 
  - используется для работы с двумерным массивом (матрицей)

- Параметр op
```char op```
  - хранит знак операции 
  - возможные значения: +, -, *

- #endif

  Завершает условную компиляцию.

  Если MATRIX_H уже определён — файл не подключится повторно.

## 8. Файл main.c


#include <stdio.h>
#include <stdlib.h>
#include <windows.h>
#include "matrix.h"

int main() {
SetConsoleOutputCP(CP_UTF8);
SetConsoleCP(CP_UTF8);

int n, i, j;
char op;

scanf("%d", &n);

double** A = (double**)malloc(n * sizeof(double*));
double** B = (double**)malloc(n * sizeof(double*));

for (i = 0; i < n; i++) {
A[i] = (double*)malloc(n * sizeof(double));
B[i] = (double*)malloc(n * sizeof(double));
}

for (i = 0; i < n; i++)
for (j = 0; j < n; j++)
scanf("%lf", &A[i][j]);

for (i = 0; i < n; i++)
for (j = 0; j < n; j++)
scanf("%lf", &B[i][j]);

scanf(" %c", &op);

double** result = calculate(A, B, n, op);

for (i = 0; i < n; i++) {
for (j = 0; j < n; j++) {
printf("%.2lf ", result[i][j]);
}
printf("\n");
}

return 0;

}

## Краткий разбор кода

- #include <stdio.h>  
  Подключает библиотеку для ввода и вывода данных (printf, scanf).

- #include <stdlib.h>  
  Подключает библиотеку для работы с динамической памятью (malloc).

- #include <windows.h>  
  Подключает библиотеку Windows для корректной работы с кодировкой.

- #include "matrix.h"  
  Подключает заголовочный файл с объявлением функции calculate.

- Функция main  
  ```int main()  ```
  главная функция программы, с неё начинается выполнение.

- Настройка кодировки  
```
SetConsoleOutputCP(CP_UTF8);  
  SetConsoleCP(CP_UTF8);  
```
обеспечивает корректный ввод и вывод русского текста.

- Переменные  
  ```
  int n, i, j;  
  char op;
  ``` 
  n — размер матриц  
  i, j — индексы циклов  
  op — знак операции

- Ввод размера  
  ```scanf("%d", &n);```  
  вводится размер квадратных матриц.

- Создание матриц  
  ```double** A = (double**)malloc(n * sizeof(double*));```
  ```double** B = (double**)malloc(n * sizeof(double*));```
  выделяется память под массив указателей (строки матриц).

- Выделение памяти для строк  
  ```
  for (i = 0; i < n; i++) {  
  A[i] = (double*)malloc(n * sizeof(double));  
  B[i] = (double*)malloc(n * sizeof(double));  
  }
  ``` 
  создаются строки матриц, формируется структура n × n.

- Ввод матрицы A  
  ```scanf("%lf", &A[i][j]); ``` 
  ввод элементов первой матрицы.

- Ввод матрицы B  
  ```scanf("%lf", &B[i][j]);  ```
  ввод элементов второй матрицы.

- Ввод операции  
  ```scanf(" %c", &op);```  
  считывается знак операции (+, -, *), пробел нужен для пропуска Enter.

- Вызов функции  
  ```double** result = calculate(A, B, n, op);  ```
  выполняется операция над матрицами, возвращается новая матрица.

- Вывод результата  
  ```printf("%.2lf ", result[i][j]);  ```
  вывод элементов матрицы с точностью до 2 знаков.

- Перенос строки  
  ```printf("\n");  ```
  переход на новую строку после каждой строки матрицы.

- return 0;  
  ```return 0; ``` 
  завершает программу успешно.
