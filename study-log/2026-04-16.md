## 1. Что такое строка в языке C

Строка — это массив символов типа char, который заканчивается специальным символом \0 (конец строки).

Пример объявления строки:

```c
char str[81];
```
Где:

str — имя строки
81 — максимальное количество символов (80 + символ конца строки)

---

# 2. Ввод строки с клавиатуры

Для ввода строки используется функция fgets.

```c
fgets(str, 81, stdin);
```
Особенности:
- считывает строку с пробелами
- ограничивает длину строки
- автоматически добавляет \0 в конец строки

Общий принцип:

```
ввести строку → сохранить в массив → работать с символами
```
---


# 3. Цикл for

Цикл for используется для перебора элементов строки.

Общий вид:

```c
for (инициализация; условие; шаг) {
    // тело цикла
}
```
В работе со строкой:

```c
for (i = 0; str[i] != '\0'; i++)
```

Где:
- i — индекс символа
- str[i] — текущий символ
- '\0' — конец строки

---

# 4. Поэлементный обход строки

Каждый символ строки обрабатывается отдельно.

Принцип:

```
берём символ → проверяем → изменяем при необходимости
```
Пример:

```c
str[0]
str[1]
str[2]
...
```

---

# 5. Условный оператор if

Используется для проверки символа.

Общий вид:

```c
if (условие) {
    // действие
}
```
В данной задаче:
```c
if (str[i] == 'a') {
    str[i] = 'A';
}
else if (str[i] == 'b') {
    str[i] = 'B';
}
```

Что происходит:
- если символ a → заменяем на A
- если символ b → заменяем на B

---

# 6. Вывод результата

Для вывода используется функция printf:

```c
if (условие) {
    // действие
}
```
Где:
- %s — строка
- \n — переход на новую строку

# 7. Работа с кодировкой (важно для Windows)

Для корректного отображения русского текста используется:

```c
SetConsoleOutputCP(CP_UTF8);
SetConsoleCP(CP_UTF8);
```
Это позволяет:
- корректно выводить кириллицу
- корректно вводить текст
