1\. Функции в языке C
---------------------

Функция --- основной структурный элемент программы

Объявление:

```
тип имя(параметры)
```

В программе используются функции:

-   `PrintNormal()`
-   `PrintReverse()`

Каждая функция выполняет отдельную задачу

* * * * *

2\. Рекурсия
------------

Рекурсия --- это вызов функцией самой себя.

Пример:

```
PrintNormal(n / 10);
```

Особенность:

-   каждый вызов создаёт новый уровень выполнения
-   используется стек вызовов

Рекурсия заменяет циклы

* * * * *

3\. Арифметические операции
---------------------------

Используются операции:

```
n / 10n % 10
```

Согласно лекциям:

-   `/` --- деление
-   `%` --- остаток от деления

Пример:

```
179 / 10 = 17179 % 10 = 9
```

* * * * *

4\. Условный оператор
---------------------

Используется:

```
if (n < 10)
```

Назначение:

-   определение базового случая рекурсии
-   остановка дальнейших вызовов

Условный оператор реализует выбор действия

* * * * *

5\. Работа с числами
--------------------

Число рассматривается как последовательность цифр.

Разделение числа:

-   последняя цифра → `n % 10`
-   оставшаяся часть → `n / 10`

Это позволяет обрабатывать число без массивов

* * * * *

6\. Разложение числа на цифры
-----------------------------

Алгоритм:

1.  отделить последнюю цифру
2.  уменьшить число
3.  повторить процесс

Пример:

```
179 → 17 → 1
```

* * * * *

7\. Вывод данных
----------------

Вывод выполняется:

```
printf("%d ", n);
```

Согласно лекциям:

-   `%d` --- вывод целого числа

* * * * *

8\. Обычный порядок вывода
--------------------------

Алгоритм:

1.  сначала вызвать рекурсию
2.  затем вывести цифру

```
PrintNormal(n / 10);printf("%d ", n % 10);
```

Результат:

```
1 7 9
```

* * * * *

9\. Обратный порядок вывода
---------------------------

Алгоритм:

1.  сначала вывести цифру
2.  затем вызвать рекурсию

```
printf("%d ", n % 10);PrintReverse(n / 10);
```

Результат:

```
9 7 1
```
