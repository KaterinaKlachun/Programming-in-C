## 1. Что такое функция

Функция — это блок кода, который выполняет определённую задачу и может возвращать значение.

Общий вид:
```
тип имя(параметры) {
// код
return значение;
}
```
---

## 2. Формулы

Периметр:
```
P = a + b + c
```
Площадь (формула Герона):
```
p = (a + b + c) / 2
S = √(p * (p - a) * (p - b) * (p - c))
```
---
## 3. Проверка треугольника

Треугольник существует, если:
```c
a + b > c
a + c > b
b + c > a
```

- #ifndef TRIANGLE_H  
  Директива условной компиляции.

- #define TRIANGLE_H  
  Если файл ещё не подключался — создаётся определение TRIANGLE_H.  
  Это предотвращает повторное подключение файла.


- Объявление функции perimeter  
  ```double perimeter(double a, double b, double c);```
  - объявление функции (прототип)
  - принимает три стороны треугольника
  - возвращает периметр


- Объявление функции area  
  ```double area(double a, double b, double c);```
  - объявление функции (прототип)
  - принимает три стороны треугольника
  - возвращает площадь


- #endif  
  Завершает условную компиляцию.  
  Если TRIANGLE_H уже определён — файл не подключится повторно.

## 6. Файл triangle.c
```c
   #include <math.h>      // подключаем библиотеку для математических функций (sqrt)
#include "triangle.h"  // подключаем заголовочный файл с объявлениями функций

double perimeter(double a, double b, double c) {
    // функция вычисляет периметр треугольника
    // принимает 3 стороны и возвращает их сумму
    return a + b + c;
}

double area(double a, double b, double c) {
    // функция вычисляет площадь треугольника по формуле Герона

    double p = perimeter(a, b, c) / 2;
    // p — полупериметр (половина периметра)

    return sqrt(p * (p - a) * (p - b) * (p - c));
    // sqrt — квадратный корень
    // формула Герона:
    // S = √(p * (p - a) * (p - b) * (p - c))
}
```
## Краткий разбор кода
- #include <math.h>
Подключает математическую библиотеку, необходимую для функции sqrt() (квадратный корень).


- #include "triangle.h"
Подключает заголовочный файл, в котором объявлены функции perimeter и area.
Это нужно, чтобы компилятор знал о существовании этих функций.


- Функция perimeter
double perimeter(double a, double b, double c)
принимает три значения (стороны треугольника)
возвращает их сумму
```формула: P = a + b + c```


- Функция area
double area(double a, double b, double c)
  - Использует формулу Герона: ```p = (a + b + c) / 2```
    ```S = √(p * (p - a) * (p - b) * (p - c))```


- Переменная p
```double p = perimeter(a, b, c) / 2;```
  - это полупериметр
  - вычисляется через уже созданную функцию perimeter
  это делает код чище и переиспользуемым


- sqrt(...)
```sqrt(...)```
  - функция из math.h
  - вычисляет квадратный корень
  - используется для нахождения площади

## 7. Файл main.c
```c
   #include <stdio.h>
   #include <windows.h>
   #include "triangle.h"

int main() {
SetConsoleOutputCP(CP_UTF8);
SetConsoleCP(CP_UTF8);

    double a, b, c;

    printf("Введите стороны треугольника:\n");
    scanf("%lf %lf %lf", &a, &b, &c);

    if (a + b > c && a + c > b && b + c > a) {
        printf("Периметр: %.2lf\n", perimeter(a, b, c));
        printf("Площадь: %.2lf\n", area(a, b, c));
    } else {
        printf("Треугольник не существует\n");
    }

    return 0;
}
```

## Пояснение

- #include <stdio.h>  
  Подключает библиотеку для ввода и вывода данных.  
  Используется для функций printf и scanf.


- #include <windows.h>  
  Подключает библиотеку Windows.  
  Используется для корректной работы с кодировкой (русский текст в консоли).


- #include "triangle.h"  
  Подключает заголовочный файл, в котором объявлены функции perimeter и area.  
  Это нужно, чтобы компилятор знал о существовании этих функций.


- Функция main  
  ```int main()``` 
  главная функция программы  
  с неё начинается выполнение кода


- Настройка кодировки  
  SetConsoleOutputCP(CP_UTF8);  
  SetConsoleCP(CP_UTF8);
    - позволяет корректно выводить русский текст
    - позволяет корректно вводить данные


- Переменные  
  ```double a, b, c;```
    - переменные для хранения сторон треугольника
    - тип double используется для вещественных чисел


- Ввод данных  
  ```scanf("%lf %lf %lf", &a, &b, &c);```
    - %lf — формат для типа double
    - & — передаёт адрес переменной
    - вводятся три стороны треугольника


- Проверка треугольника  
  ```a + b > c && a + c > b && b + c > a```
    - проверяется условие существования треугольника
    - сумма любых двух сторон должна быть больше третьей


- Вызов функции perimeter  
  ```perimeter(a, b, c)```
    - вычисляет периметр
    - формула: P = a + b + c


- Вызов функции area  
  ```area(a, b, c)```
    - вычисляет площадь
    - формула Герона: p = (a + b + c) / 2
    - S = √(p * (p - a) * (p - b) * (p - c))


- Вывод результата  
  ```
  printf("Периметр: %.2lf\n", ...);  
  printf("Площадь: %.2lf\n", ...);
  ```
    - %.2lf — вывод числа с точностью до 2 знаков
    - \n — перенос строки


- Обработка ошибки  
  ```printf("Треугольник не существует\n");```
    - выводится, если треугольник не существует


- return 0;  
  ```return 0;```
    - завершает программу
    - означает успешное выполнение
