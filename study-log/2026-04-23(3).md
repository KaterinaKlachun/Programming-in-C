## 1. Что такое текстовый файл

Текстовый файл — это файл, в котором данные хранятся в виде строк символов.

В нашей задаче каждая строка файла содержит:
- фамилию
- имя
- отчество
- год рождения

Пример строки:
Иванов Петр Сергеевич 1975

Такой формат удобен тем, что его можно читать по частям: сначала три строки типа string, потом одно число.

---

## 2. Что такое файловый указатель

Для работы с файлами в C используется специальный тип: ```c **FILE** *f;```

Файловый указатель — это переменная, через которую программа работает с файлом.

В этой задаче нужно два файловых указателя:
- один для чтения исходного файла
- второй для записи результата

---

## 3. Открытие файла

Файл открывается функцией:

```fopen(...)```

В этой задаче используются два режима:

- Чтение:
  ```
  fopen(*input.txt*, *r*)
  ```
- Запись:
    ```
    fopen(*output.txt*, *w*)
    ```

Что это значит:

*r* — открыть файл для чтения

*w* — открыть файл для записи

---

## 4. Чтение данных из файла по шаблону

Так как каждая строка имеет один и тот же формат: Фамилия Имя Отчество Год

удобно использовать чтение по шаблону:

```fscanf(in, *%s %s %s %d*, surname, name, patronymic, &year)```

Что здесь происходит:

- %s — считывает строку до пробела
- %d — считывает целое число
- surname, name, patronymic — массивы символов
- year — переменная типа int

То есть одна команда сразу считывает всю запись из строки файла.

---

5\. Цикл чтения файла
---------------------

Файл читается не один раз, а пока в нём есть данные.

Для этого удобно использовать цикл:

```while (fscanf(...) == 4)```

Почему именно `== 4`:

-   ожидается 4 элемента
-   фамилия
-   имя
-   отчество
-   год рождения

Если считались все 4 значения, значит строка корректная и можно её обрабатывать дальше.

6\. Проверка условия
--------------------

По условию нужно оставить только тех, кто родился позднее 1980 года.

Проверка:

```if (year > 1980)```

Если условие выполняется, строка записывается в выходной файл.

* * * * *

7\. Запись в другой файл
------------------------

Для записи используется функция:

```fprintf(...)```

Пример:

```fprintf(out, "%s %s %s %d\n", surname, name, patronymic, year);```

Она записывает данные в файл в том же формате, что и были в исходном.

* * * * *

8\. Закрытие файлов
-------------------

После окончания работы файлы нужно обязательно закрыть:

```
fclose(in);\
fclose(out);
```

Это нужно для корректного завершения работы программы и сохранения данных.

11\. Файл main.c
---
```
#include <stdio.h>\
#include <windows.h>

int main() {\
SetConsoleOutputCP(CP_UTF8);\
SetConsoleCP(CP_UTF8);

    FILE *in, *out;

    char surname[50], name[50], patronymic[50];\
    int year;

    in = fopen("input.txt", "r");\
    out = fopen("output.txt", "w");

    if (in == NULL || out == NULL) {\
        printf("Ошибка открытия файла\n");\
        return 1;\
    }

    while (fscanf(in, "%s %s %s %d", surname, name, patronymic, &year) == 4) {\
        if (year > 1980) {\
            fprintf(out, "%s %s %s %d\n", surname, name, patronymic, year);\
        }\
    }

    fclose(in);\
    fclose(out);

    printf("Готово. Результат записан в output.txt\n");

    return 0;\
}
```

    массивы символов для хранения фамилии, имени и отчества
-   `int year;`\
    переменная для хранения года рождения

    открывает исходный файл для чтения
-   `out = fopen("output.txt", "w");`\
    открывает выходной файл для записи

    проверка, удалось ли открыть файлы\
    если нет --- программа завершится с ошибкой
-   `while (fscanf(in, "%s %s %s %d", surname, name, patronymic, &year) == 4)`\
    цикл чтения файла\
    каждая итерация считывает одну строку
-   `if (year > 1980)`\
    проверка условия задачи\
    оставляем только тех, кто родился позже 1980 года
-   `fprintf(out, "%s %s %s %d\n", surname, name, patronymic, year);`\
    записывает подходящую строку в выходной файл
-   `fclose(in);`\
    закрывает входной файл
-   `fclose(out);`\
    закрывает выходной файл
-   `return 0;`\
    завершает программу успешно
