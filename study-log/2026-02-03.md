## 1. Что такое массив в языке C

Массив — это структура данных, предназначенная для хранения  
нескольких значений **одного типа** под одним именем.

Пример одномерного массива:
```c
int arr[5];
```
Это означает:
- arr — имя массива 
- 5 — количество элементов 
- каждый элемент имеет тип int

Обращение к элементу массива происходит по индексу:

```c
arr[0]
arr[1]
```
*⚠️ Нумерация элементов начинается с нуля.*

---
## 2. Двумерный массив (матрица)

Двумерный массив — это массив массивов, который удобно представлять в виде таблицы (матрицы).

Пример объявления матрицы 3×3:
```c
double matrix[3][3];
```
Где:
- первый индекс — номер строки 
- второй индекс — номер столбца

---

## 3. Ввод элементов матрицы с клавиатуры

Для ввода элементов матрицы используются вложенные циклы ```for```.

Общий принцип:
```text
для каждой строки
    для каждого столбца
        ввод элемента
```
Где:
- первый индекс — номер строки
- второй индекс — номер столбца

---

## 4. Цикл ```for```

Цикл ```for``` используется для многократного выполнения кода.

Общий вид:
```c
for (инициализация; условие; шаг) {
    // тело цикла
}
```
Пример:
```
for (int i = 0; i < 3; i++) {
// код выполнится 3 раза
}
```
---

## 5. Главная диагональ матрицы

Главная диагональ — это элементы, у которых:
```номер строки == номер столбца```

Примеры элементов главной диагонали:
```text
matrix[0][0]
matrix[1][1]
matrix[2][2]
```
Сумма элементов главной диагонали находится при условии:
```
if (i == j)
```

---

## 6. Побочная диагональ матрицы

Побочная диагональ — это элементы, у которых:
```i + j == размер_матрицы - 1```

Примеры для матрицы 3×3:
```text
matrix[0][2]
matrix[1][1]
matrix[2][0]
```

---

## 7. Побочная диагональ матрицы

Для работы с матрицами любого размера:
- размер матрицы вводится с клавиатуры 
- используется один и тот же код с циклами for

Пример:

```c
int n;
scanf("%d", &n);
double matrix[n][n];
```
