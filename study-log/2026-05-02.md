1\. Обработка сигналов
----------------------

Сигнал --- это уведомление от операционной системы программе о событии.

Примеры сигналов:

-   `SIGINT` --- прерывание с клавиатуры
-   `SIGSEGV` --- ошибка доступа к памяти
-   `SIGTERM` --- завершение программы

Программа может:

-   игнорировать сигнал
-   обработать сигнал
-   завершиться

В данной работе используется **обработка сигналов через пользовательскую функцию**

* * * * *

2\. Функция-обработчик
----------------------

Обработчик объявляется как обычная функция:

```
void SignalHandler(int sig)
```

Особенности:

-   параметр `sig` --- номер сигнала
-   тип `void` --- ничего не возвращает

Это соответствует лекциям, где функция --- основной элемент программы

* * * * *

3\. Регистрация обработчиков
----------------------------

Связывание сигнала с функцией:

```
signal(SIGINT, SignalHandler);
```

Что происходит:

-   ОС при получении сигнала передаёт управление функции
-   стандартное поведение заменяется пользовательским

* * * * *

4\. Генерация сигналов
----------------------

Для тестирования используется функция:

```
raise(SIGSEGV);
```

Она:

-   искусственно вызывает сигнал
-   позволяет проверить обработчик

* * * * *

5\. Использование switch
------------------------

В обработчике применяется:

```
switch(sig)
```

Согласно лекциям, `switch` используется для выбора одного из вариантов

Принцип работы:

-   проверяется значение `sig`
-   выбирается соответствующий case
-   выполняется нужный код

* * * * *

6\. Консольный ввод и вывод
---------------------------

Вывод:

```
printf(...)
```

Ввод:

```
scanf("%d", &choice);
```

В лекциях указано:

-   `%d` --- целое число
-   `&` --- передача адреса переменной

* * * * *

7\. Цикл do while
-----------------

Меню реализовано через:

```
do { ... } while (choice != 0);
```

Особенность:

-   выполняется минимум один раз
-   затем проверяется условие

Это соответствует описанию цикла в лекциях

* * * * *

8\. Меню программы
------------------

Меню --- это последовательность команд:

-   вывод списка действий
-   ввод выбора
-   выполнение действия

Реализовано через `printf` и `switch`

* * * * *

9\. Логирование
---------------

Логирование --- это вывод информации о событиях программы.

В данной работе:

```
printf("Получен сигнал...");
```

Выводится:

-   тип сигнала
-   описание
